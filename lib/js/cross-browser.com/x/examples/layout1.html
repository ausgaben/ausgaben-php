What

This is an example of a dynamic, downgradable DHTML layout.

Disable javascript and reload this page to see it in its 'downgrade mode'. You can also downgrade based on the absence of certain browser objects.

How

Six files are required by this page:

  1. layout1.html - the HTML for the page.
  2. layout1.js - the javascript that sizes and positions all major container elements on the page.
  3. layout1d.css - the default styles for the page, which are also the 'downgrade mode' styles.
  4. layout1x.css - styles which over-ride the default styles (non-downgrade). This file only gets included if javascript is enabled.
  5. 'x_core.js' and 'x_event.js' - the DHTML library files used by the code in layout.js. These files only get included if javascript is enabled.
