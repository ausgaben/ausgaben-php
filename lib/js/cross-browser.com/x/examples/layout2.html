What Is This?

This demo started life with Kevin Yank's SitePoint Article. Kevin does an excellent job of explaining the design and implementation of this page. I modified Kevin's demo and now provide it here at Cross-Browser.com. The following are some of the features of this demo.

  • It uses CSS for the basic 3-column layout, then uses the X library to equalize column heights onLoad and onResize.
  • It is downgradeable. Disable Javascript in your browser and refresh this page to see the downgrade layout. The page will also downgrade in the absence of required Javascript objects.
  • All Javascript, CSS, and HTML are separated into their own files.
  • It uses valid XHTML and CSS.

Implementation Tips

Most of the problems people have when trying to use this demo is that they don't read Kevin's article or don't follow his instructions.

It is important to remember that the column containers must contain statically positioned content containers.

Always develop a page in this order: (1) HTML for basic structure; (2) CSS for styling and layout; (3) Javascript for enhancements. Get the page working well at each step before proceeding to the next step.

If you have problems, post your questions at the SitePoint Javascript Forum.
